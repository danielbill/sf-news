# 想法: 奇点研究员功能

**创建日期**: 2026-02-02
**重要度**: 3 (3=最重要)
**状态**: 💡已确认

## 描述
奇点前沿项目的核心功能，每日主动运行，采集多种数据，分析新闻，给出奇点前沿的真实发展变化。

## 动机

### 问题域
该功能要解决的核心问题：
- 用户需要看清奇点实体的完整发展脉络（从奇点到全产业链）
- 需要定量分析工具来评估奇点对经济的实际影响
- 需要预测奇点未来1-3年对文明的改变
- 从定性到定量，用数字模型解决投资痛点

### 目标用户
- 项目创建者（主要用户）
- 付费订阅用户

**使用场景**：后台自动执行的程序+智能体，是看清世界脉动的透镜

### 核心价值
**核心理念**：通过分析奇点实体的战略动态和口号响应落地情况，推导出世界前进的脚步，预测出未来1-3年的文明变化。

**经济模型**：将奇点变化纳入经济模型计算，推导它对世界GDP的带动情况。

**验收标准**：用户通过该"奇点研究员"的输出，能：
1. 看清某个奇点的完整发展情况（从奇点到全产业链）
2. 看到奇点从idea到实体战略、国际经济政策出台，投资落地，大型项目落地
3. 看到GDP拉动效应
4. 估算出奇点核心实体未来价值的增长速度

---

## 功能模块拆分

### 第一组：数据基础层

| 模块 | 功能 |
|------|------|
| **模块 1: 奇点档案数据库** | Legend 实体的基础信息存储和维护（人物/组织档案 CRUD） |

---

### 第二组：外部数据采集层

| 模块 | 功能 |
|------|------|
| **模块 2: 财经数据同步** | 同步股价、市值、营收等财务数据（集成 flaskSite1） |

---

### 第三组：情报分析层

| 模块 | 功能 |
|------|------|
| **模块 3: 战略口号跟踪** | 提取并跟踪奇点实体的战略、口号及落地情况 |
| **模块 4: 赛道竞争分析** | 分析所处赛道、竞争对手、市场格局变化 |
| **模块 5: 供应链监测** | 追踪上游供应商、下游应用的变化 |
| **模块 6: 项目进展追踪** | 追踪大型项目的落地动态 |
| **模块 7: 国策配套分析** | 分析各国政策对奇点的支持情况 |
| **模块 8: 全球备战态势** | 分析全球范围内的竞争/合作态势 |
| **模块 9: 观点情绪聚合** | 汇总各方对奇点的观点和态度 |

---

### 第四组：经济模型层

| 模块 | 功能 |
|------|------|
| **模块 10: 估值模型** | 基于财务和动态数据估算奇点实体价值 |
| **模块 11: GDP 拉动计算** | 计算对国家/全球 GDP 的影响 |
| **模块 12: 增长速度预测** | 预测未来价值增长速度 |

---

**总计: 12 个独立功能模块**

## 数据采集手段

奇点研究员可重复使用所有手段进行数据采集，包括但不限于：

| 手段 | 用途 |
|------|------|
| `/baidu-ai-search` | 智能信息聚合采集 |
| `/github-kb` | 在GitHub上找到符合需求的各种工具 |
| `flaskSite1` 项目 | 财经数据采集接口及数据库 |

## 依赖关系

**技术依赖**：
- 本项目中的**新闻采集业务**（核心依赖）
- 财务数据模块（专业财经API）
- 公司档案模块（手动维护，AI辅助）

**业务依赖**：
- Legend实体库需要有一定数据积累
- 关键词配置系统
- 数据存储和处理管道

**未来扩展**：
- 关联分析模块（后期实现，智能关联）
- 用户订阅和付费系统
- 可视化Dashboard
