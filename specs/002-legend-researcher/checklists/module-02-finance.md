# 模块 2: 财经数据同步

**层级**: 外部数据采集层
**状态**: 待实现
**优先级**: P1

## 功能描述

同步股价、市值、营收等财务数据，集成 flaskSite1 项目的财经数据采集接口。

## 验收标准

- [ ] 能够从 flaskSite1 获取财务数据
- [ ] 支持 Legend 关联公司的股价/市值数据同步
- [ ] 支持定期自动更新（日级别）
- [ ] 数据存储到本地数据库
- [ ] 提供数据查询接口供其他模块使用

## 依赖

- 依赖模块 1（奇点档案数据库）获取公司信息

## 输出物

- `sync/finance_sync.py` - 财经数据同步脚本
- `models/finance.py` - 财务数据模型
- `tests/test_finance_sync.py` - 单元测试
