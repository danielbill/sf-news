# 奇点档案设计理念

## 核心理念

**Legend = 追逐实体**

- 奇点可以是**人物**（PERSON），也可以是**公司/组织**（ORGANIZATION）
- 公司是独立生命体，有些决策已脱离人类范畴（AI参与决策）
- 领袖的角色是**确保组织走在正确道路上**
- 奇点往往来自**小创新的指数级爆发**

---

## 奇点与超新星

### SINGULARITY（奇点）- 既定事实

**定义**：已经开启新时代，全球公认，不可逆

奇点 = 必然影响所有人，这是定义本身，不是评分结果。

| 实体 | 开启的时代 | 证据 |
|------|-----------|------|
| **黄仁勋 / NVIDIA** | AI 算力时代 | GPU 成为时代基础设施，没有 GPU 就没有这个时代 |
| **马斯克** | 太空时代 + 电动车时代 | 星舰/星链、特斯拉全球普及，不可逆 |

### SUPERNOVA（超新星）- 爆发中，未确定

**定义**：正在爆发式成长，极有可能成为奇点，但也可能快速陨落

超新星 = **还未完成**那个定义时代的成就，但能量巨大值得追踪。

| 实体 | 爆发证据 | 能否成为奇点取决于 |
|-----------|----------|-------------------|
| **Sam Altman / OpenAI** | ChatGPT 全球现象 | AGI 是否真的实现、是否安全 |
| **Dario Amodei / Anthropic** | Claude 快速追赶 | 能否在 AGI 竞赛中胜出 |
| **字节跳动** | TikTok 统治级 | 能否从内容平台跃迁为时代定义者 |

---

## 时代的本质

**奇点的本质是开启新时代**

```
发明电 → 开启电气时代 → 影响所有人
蒸汽机 → 开启工业时代 → 影响所有人
GPU → 开启 AI 算力时代 → 影响所有人
火星殖民 → 开启太空时代 → 影响所有人（进行中）
AGI → 开启智能时代 → 影响所有人（未来）
```

### 三个层次的真正区别

| 层次 | 定义 | 例子 |
|------|------|------|
| **SINGULARITY** | 开启**新时代**（从0到1，创造新范式） | 电气时代、AI 算力时代、太空时代 |
| **QUASI** | ~~在时代内改变行业~~ | 已取消 |
| ~~POTENTIAL~~ | ~~有潜力~~ | 已取消 |

**不再使用** QUASI 和 POTENTIAL，只保留：
- **SINGULARITY（奇点）** — 既定事实
- **SUPERNOVA（超新星）** — 爆发中

---

## 追人 vs 追公司

**追逐实体** = 追人 + 追公司

- 人是变量，公司是载体
- 公司是独立生命体，有演化逻辑
- 领袖是方向校准器，确保组织走在正确道路上
- 小创新 + 正反馈放大 = 奇点爆发
